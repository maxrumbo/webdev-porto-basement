Project

  • Simple Todos Manager in C language

    This project is a simple command-line activity management solution. Designed as a final project for a Procedural Programming course, it focuses on stream operations to read and write activity data from a text file. Its main features include loading initial data, adding new activities with automatic IDs, updating activity statuses, and displaying the entire activity list. The project demonstrates the ability to efficiently manage data through a standard input/output (STDIN/STDOUT) interface. Check it on GitHub.
